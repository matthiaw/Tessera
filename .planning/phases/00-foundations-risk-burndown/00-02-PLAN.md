## Image pinning

The Apache AGE container is **pinned by sha256 digest**, not by floating tag, per ADR / Phase 0 D-09:

```
apache/age@sha256:<DIGEST_FROM_TASK_1>
```

This digest appears in three enforcement sites — bumping it requires updating all three:

1. `docker-compose.yml` (this file)
2. `fabric-core/src/test/java/dev/tessera/core/support/AgePostgresContainer.java` (Testcontainers helper, plan 03)
3. This README section

To resolve the current digest:

```bash
docker pull apache/age:PG16_latest
docker image inspect --format '{{index .RepoDigests 0}}' apache/age:PG16_latest
```
