**SqlViewNameResolver** (pure utility, no Spring):
- `public static String resolve(UUID modelId, String typeSlug)` — takes model UUID and type slug, returns `v_{first8HexOfUUID}_{slug_underscored}`. Replace hyphens with underscores. Validate result length <= 63 chars; if exceeded, truncate typeSlug portion and append hash suffix. Throw IllegalArgumentException on null/empty.

**SqlViewProjection** (@Component in `dev.tessera.projections.sql`):
- Constructor: inject `SchemaRegistry`, `SchemaVersionService`, `NamedParameterJdbcTemplate`
- `public void regenerateForTenant(TenantContext ctx)`:
  1. Call `schemaRegistry.listNodeTypes(ctx)` to get all NodeTypeDescriptors
  2. For each type, query AGE label catalog: `SELECT l.name, n.nspname FROM ag_catalog.ag_label l JOIN ag_catalog.ag_graph g ON l.graph = g.graphid JOIN pg_namespace n ON n.oid = l.relation::regclass::oid WHERE g.name = :graph_name AND l.kind = 'v'` to resolve the actual PostgreSQL schema + table name for the label
  3. Generate DDL: `CREATE OR REPLACE VIEW {viewName} AS /* schema_version:{ver} model_id:{id} type:{slug} */ SELECT (properties::jsonb)->>'uuid' AS uuid, (properties::jsonb)->>'model_id' AS model_id, {columnList} FROM "{schema}"."{LabelName}" WHERE ((properties::jsonb)->>'model_id')::uuid = '{tenantId}'::uuid AND COALESCE((properties::jsonb)->>'_tombstoned', 'false')::boolean = false`
  4. CRITICAL: Use `(properties::jsonb)->>'key'` NOT `properties->>'key'` — agtype is NOT jsonb (Pitfall 1)
  5. Execute DDL via `jdbc.getJdbcTemplate().execute(ddl)`
  6. Track active views in a `ConcurrentHashMap<String, ViewMetadata>` for the admin endpoint
- `public void regenerateAll()`: iterate all known tenants and call regenerateForTenant for each
- Implement `ApplicationRunner` to call `regenerateAll()` on startup (per D-A2)
- Implement `@EventListener` or `@TransactionalEventListener` for schema change events — check if SchemaRegistry publishes an ApplicationEvent; if not, create a simple `SchemaChangedEvent` record and publish it from SchemaRegistry after mutations. Listener calls `regenerateForTenant(ctx)`.
- `public List<ViewMetadata> listViews(UUID modelId)` — returns view metadata for admin endpoint

**Column type mapping** from PropertyDescriptor.dataType():
- STRING → `(properties::jsonb)->>'slug' AS slug` (text)
- INTEGER → `((properties::jsonb)->>'slug')::integer AS slug`
- BOOLEAN → `((properties::jsonb)->>'slug')::boolean AS slug`
- TIMESTAMP → `((properties::jsonb)->>'slug')::timestamptz AS slug`
- REFERENCE → `((properties::jsonb)->>'slug')::uuid AS slug` (UUID reference)
- Default/unknown → text cast via `->>`
