```java
package dev.tessera.projections.mcp.interceptor;

/**
 * Wraps every MCP tool response in {@code <data>...</data>} markers to mitigate
 * prompt injection (SEC-08, D-D1). Applied ONLY by SpringAiMcpAdapter — individual
 * tools must NOT call this directly.
 */
public final class ToolResponseWrapper {
    private static final String OPEN = "<data>";
    private static final String CLOSE = "</data>";

    private ToolResponseWrapper() {}

    public static String wrap(String rawContent) {
        if (rawContent == null) return OPEN + CLOSE;
        return OPEN + rawContent + CLOSE;
    }
}
```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Agent -> MCP SSE | Untrusted LLM agent sends tool invocations over SSE; JWT authenticates but agent content is untrusted |
| MCP -> GraphRepository | MCP tools pass agent-provided parameters (Cypher, UUIDs, filters) to graph queries |
| MCP -> Response to Agent | Source data from graph may contain adversarial content that could manipulate the agent |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Spoofing | SpringAiMcpAdapter | mitigate | JWT validation via RotatableJwtDecoder on every request; tenant extracted from signed claim, not user input |
| T-03-02 | Tampering | GraphRepository.executeTenantCypher | mitigate | Mutation keyword blocklist (DELETE, CREATE, MERGE, SET, REMOVE, DROP) rejects write attempts; model_id injected server-side |
| T-03-03 | Information Disclosure | findShortestPath | mitigate | ALL(n IN nodes(path) WHERE n.model_id) filter prevents cross-tenant path traversal |
| T-03-04 | Elevation of Privilege | SecurityConfig | mitigate | ROLE_AGENT cannot access /admin/** (blocked by hasAnyRole("ADMIN", "TOKEN_ISSUER") rule) |
</threat_model>
