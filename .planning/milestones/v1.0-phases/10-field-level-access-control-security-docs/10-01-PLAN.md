**AclPropertyCache** (per D-08):
Create `fabric-core/src/main/java/dev/tessera/core/security/AclPropertyCache.java`:
- Package: `dev.tessera.core.security`
- Annotate with `@Component`
- Apache 2.0 license header (copy from EncryptionStartupGuard.java)
- Private Caffeine cache: `Cache<AclCacheKey, Set<String>> cache`
- Cache config: `maximumSize(1_000)`, `expireAfterAccess(Duration.ofMinutes(30))`, `recordStats()`
- Public record `AclCacheKey(UUID modelId, String typeSlug, String canonicalRoleSet)` -- canonicalRoleSet is the caller's roles sorted alphabetically and comma-joined (e.g., `"ADMIN,AGENT"`) to avoid Pitfall 1 (role set ordering)
- Static method `canonicalizeRoles(Set<String> roles)` returns `roles.stream().sorted().collect(Collectors.joining(","))` -- empty set produces `""`
- Method `getAllowedPropertySlugs(UUID modelId, String typeSlug, Set<String> callerRoles, Supplier<List<PropertyDescriptor>> propertyLoader)`:
  1. Compute `canonicalRoleSet = canonicalizeRoles(callerRoles)`
  2. `cache.get(new AclCacheKey(modelId, typeSlug, canonicalRoleSet), k -> computeAllowed(callerRoles, propertyLoader.get()))`
  3. `computeAllowed()`: iterate properties, include slug if `property.readRoles()` is null or empty (D-02) OR if `callerRoles` intersects `property.readRoles()` (use `!Collections.disjoint(callerRoles, property.readRoles())`)
  4. Return unmodifiable `Set<String>`
- Method `invalidateAll(UUID modelId)`: `cache.asMap().keySet().removeIf(k -> k.modelId().equals(modelId))`
- Method `invalidateAll()`: `cache.invalidateAll()` (for tests)

- Constructor injection: `AclPropertyCache cache`
- Method `filterProperties(NodeState node, NodeTypeDescriptor descriptor, Set<String> callerRoles)` returns `Map<String, Object>`:
  1. Get `Set<String> allowedSlugs = cache.getAllowedPropertySlugs(descriptor.modelId(), descriptor.slug(), callerRoles, descriptor::properties)`
  2. Build `LinkedHashMap<String, Object> filtered`
  3. Iterate `node.properties().entrySet()`, include entry if `allowedSlugs.contains(entry.getKey())`
  4. Return `filtered` (empty map if all redacted -- per D-07)
- Method `isTypeVisible(NodeTypeDescriptor descriptor, Set<String> callerRoles)` returns `boolean`:
  1. If `descriptor.readRoles()` is null or empty, return `true` (D-02 semantics)
  2. Else return `!Collections.disjoint(callerRoles, new HashSet<>(descriptor.readRoles()))`
- Method `checkWriteRoles(NodeTypeDescriptor descriptor, Map<String, Object> payload, Set<String> callerRoles)` throws `AccessDeniedException`:
  1. For each key in `payload.keySet()`, find matching `PropertyDescriptor` in `descriptor.properties()`
  2. If property has non-empty `writeRoles()` and `callerRoles` does not intersect, throw `new AccessDeniedException("Caller lacks write role for property '" + key + "'")`
  3. Use existing Spring Security `AccessDeniedException` or a custom domain exception that the ControllerAdvice maps to 403
