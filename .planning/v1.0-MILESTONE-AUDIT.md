## Cross-Phase Integration Gaps

### 1. TesseraMetrics never called by production code
`TesseraMetrics.recordIngest()`, `recordRuleEvaluation()`, `recordConflict()`, and `shaclTimer()` are registered and tested in isolation but never injected into or called by `ConnectorRunner`, `GraphServiceImpl`, or `ShaclValidator`. Prometheus counters stay at zero.
**Affects:** OPS-01

### 2. SchemaChangeEvent does not exist
`SchemaRegistry.createNodeType()` persists to DB but publishes no `ApplicationEvent`. `SqlViewProjection` has no `@EventListener` — views go stale after runtime type registration. `SpringAiMcpAdapter.notifySchemaChanged()` is public but uncalled — MCP clients not notified.
**Affects:** SQL-02, MCP-08

### 3. Circlead ConnectorInstance not wired
`CircleadConnectorConfig` exposes three `MappingDefinition` beans. No code constructs `ConnectorInstance` objects from them and registers with `ConnectorRegistry`. `ConnectorScheduler.tick()` returns empty — circlead sync never runs in production.
**Affects:** CIRC-01, CIRC-02

### 4. Spring Cloud Vault missing from classpath
`application-prod.yml` references `spring.config.import=vault://` but `spring-cloud-starter-vault-config` is absent from all compile-scope POMs. Vault health indicator impossible. Secrets must come from env vars.
**Affects:** SEC-02, OPS-02
